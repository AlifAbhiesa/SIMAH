# SIMAH

Sistem Informasi Keanggotaan Mahasiswa
 
<h1>Feature :</h1>
1. Master Data
- Anggota
- Hukuman
- AD/ART
- Kegiatan
- Proker
2. Kepanitiaan
3. Pelanggaran
4. Prestasi

<h1> Install </h1>
1. Open NetBeans IDE
2. Open Project
3. Load This Project
4. Right Click on Libraries, choose Add Folder/Jar
5. Choose jlx and jTattoo from project
6. Load simah.sql into your database server
